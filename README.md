Project Title : 
Interactive Sales Dashboard using Microsoft Excel

Project Description : 
This project focuses on developing an interactive and dynamic sales dashboard using Microsoft Excel to analyze and visualize business performance data. The dashboard provides meaningful insights into sales trends, regional performance, product-wise contribution, and overall profitability.

The system uses structured sales data and transforms it into visual reports through pivot tables and charts. Interactive elements such as slicers and timelines are integrated to allow users to filter data by region, category, and time period, making the dashboard user-friendly and decision-oriented.

The dashboard helps in identifying key business insights such as top-performing products, high-revenue regions, and seasonal sales patterns. This project demonstrates practical knowledge of data analysis, visualization, and reporting using Excel without the need for advanced tools like Power BI.

Tools & Technologies Used : 
Microsoft Excel

Data cleaning and preprocessing

Pivot Tables for summarizing data

Charts (Bar, Line, Pie, Column) for visualization

Slicers and Timeline for interactivity

Formulas (SUM, IF, VLOOKUP/XLOOKUP)

Conditional Formatting for highlighting insights

Key Features : 
Interactive filtering using slicers

Real-time data visualization

KPI indicators (Total Sales, Profit, Orders)

Region-wise and product-wise analysis

Monthly sales trend tracking

User-friendly dashboard layout
Outcome : 
The project successfully converts raw sales data into actionable insights, helping users make better business decisions.
It enhances analytical thinking and demonstrates the ability to build professional dashboards using Excel.
